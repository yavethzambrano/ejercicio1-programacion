Elementos de un sistema de informacion computacional

1.1 Equipo Computacional

Equipo Computacional:Hardware necesario para el sistema de informacion funcione, intregrada por computadoras y equipo periferico conectado a ellas.

1.2 Recurso Humano

Recurso Humano:Son las personas que interactuan con el sistema de informacion, suministran datos o utilizan el sistema para generar datos.

1.3 Fuente

Fuente:Son los datos e iformacion que se introducen en el sistema, es decir, son las entradas requeridas para que el sistema informatico funcione.

1.4 Programas

Programas:Software ejecutado por la computadora, el cual procesa la informacion de entrada para generar los resultados esperados.

1.5 Telecomunicaciones

Telecomunicaciones:Abarca el software y hardware que permite la transmision en forma electronica de texto e imagenes, entre otros.

1.6 Politicas y Reglas de Operacion

Politicas y Reglas de Operacion:Apliacdas a los procedimientos y mecanismos para hacer que una aplicacion de computadora trabaje de acuerdo con estas o regir la parte funcional del proceso administrativo de la empresa.
